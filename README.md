hawthorne-toolbelt
==================

Command-line utility for running the Hawthorne Haus.

Supported

Planned commands:
* auth:login
* auth:logout
* social:email
* social:tweet
* lights:lock-color { #RRGGBB }
* music:on
* music:off
* music:mode { random | loop | loop-all }
* music:next
* music:prev
* music:pause
* music:play
* music:status
